# RiyadhPlacesAnalysis
Is a data analysis and visualization tool designed to provide insights into the diverse and vibrant city of Riyadh.
Data Collection:
The data of the project was from OpenStreetMap as an osm file. It was read and converted into a csv file using osmium library in Python.
Who is this project for?
Tourists and Explorers: Discover interesting places to visit and learn about the culture and history of Riyadh.
